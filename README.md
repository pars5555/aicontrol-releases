# AI Control Releases

Public download repository for [AI Control](https://aicontrol.pc.am) — AI-powered desktop and mobile agent controlled by Claude.

Source code lives in private repos — only binaries are published here.

## Downloads (always latest)

- **Windows**: [AiControl-Windows.zip](https://github.com/pars5555/aicontrol-releases/releases/latest/download/AiControl-Windows.zip) — self-contained `.NET 8` single-file build (win-x64, ~195 MB)
- **Android**: [AiControl.apk](https://github.com/pars5555/aicontrol-releases/releases/latest/download/AiControl.apk) — signed release APK, arm64-v8a + armeabi-v7a (~142 MB)

## System Requirements

**Windows**:
- Windows 10 22H2 or later
- x64 architecture

**Android**:
- Android 12 (SDK 31) or later
- arm64-v8a or armeabi-v7a

## Installation

**Windows**: Download the ZIP, extract anywhere, run `AiControl.exe`. No installer — unpacks in place.

**Android**: Download the APK, enable "Install from unknown sources" for your browser, tap the APK to install. Not on Play Store.

## Account Required

Both apps need an account at [aicontrol.pc.am](https://aicontrol.pc.am) (sign in with GitHub or Google). Billing is prepaid-credit based.
